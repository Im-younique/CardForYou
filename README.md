# CardForYou

일상생활 속 카드를 사용할 때 받을 수 있는 최대의 혜택의 카드를 찾아주는 app

### CoreFlow
1. 사용자 인증 (로그인)
2. 사용자 소유의 카드 목록 조회
3. 사용처 검색
4. 내 카드 목록 중 사용처에서 사용시 가장 많은 혜택을 받을 수 있는 카드 조회
5. 해당 카드 사용시 어느정도 혜택을 받을 수 있는지 시각화

+) 카드 목록 수동 추가/삭제  
+) 더 많은 혜택을 받을 수 있는 카드 추천

### 제약조건
카드실적이 필요한 경우, 실적은 모두 채운 것으로 가정
